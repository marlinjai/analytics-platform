# Chrome Web Store Listing

## Name
Lumitra Analytics — Heatmap Overlay

## Short Description (132 chars max)
See where users click on any website. Element-based heatmaps, scroll depth, rage click detection — powered by your Lumitra dashboard.

## Detailed Description
Lumitra Analytics brings your heatmap data directly onto any website you're analyzing. No more switching between tabs — see click patterns, scroll depth, and rage clicks overlaid on the actual page.

**What it does:**

- Click Heatmaps — See exactly where users click with beautiful radial gradient overlays. Elements are highlighted based on real click data from your Lumitra dashboard, adapting responsively to any viewport.

- Scroll Depth — Visualize how far users scroll with a color-coded gradient strip showing 25th, 50th, 75th, and 90th percentile scroll depths.

- Rage Click Detection — Identify frustrated users. Elements with rapid repeated clicks are highlighted with pulsing red outlines and click counts.

- Element Inspector — Hover over any element to see its exact click count and unique session count in a tooltip.

- Heatmap Controls — Adjust radius, opacity, and blur to tune the visualization for your data volume.

- Side Panel — Persistent analytics panel showing top clicked elements, project controls, and date/device filters alongside any page.

**How it works:**

1. Connect to your Lumitra Analytics dashboard (analytics.lumitra.co)
2. Navigate to any tracked website
3. Click "Show Heatmap" — the overlay renders directly on the page
4. Switch between Clicks, Scroll, and Rage modes
5. Use the Side Panel for persistent analytics while browsing

**Requirements:**
- A Lumitra Analytics account (analytics.lumitra.co)
- At least one tracked website with the Lumitra tracker installed

**Privacy:**
- No data is collected by the extension itself
- All analytics data comes from your own Lumitra dashboard
- The extension only communicates with analytics.lumitra.co
- No third-party tracking or ads
